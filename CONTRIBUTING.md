# Working on this repo

**Canonical repo:** `chsmith57-cell/connor-smith-site` (public).
**Live:** https://chsmith57-cell.github.io/connor-smith-site/ — GitHub Pages
auto-republishes from `main` (repo root) on every push, ~1–2 min.
**Canonical local copy:** `~/Documents/connor-smith-site`

This repo is shared by Connor, Claude Code, and Codex. To avoid overwriting
each other's work:

- **Pull first:** `git pull --rebase origin main` before starting any change.
- Commit directly to `main`, in small intentional commits; push promptly.
- **Never force-push `main`.**
- Static site, no build step — just push and Pages redeploys.
- **No credentials or private material** in this public repo (keep API keys,
  `.env*`, etc. out of it).

Project brief and design guardrails live in `PLAN.md` — read it before changing
the visuals.
